Markov Algorithm Calculator

Алгоритмы Маркова представляют собой формальные системы, которые используются для описания вычислений и манипуляций с символами. Они основаны на правилах, которые определяют, как преобразовывать строки символов, и находят применение в различных областях, включая теорию автоматов и логику.

Правила алгоритмов Маркова записываются в виде пар, где каждая пара состоит из строки, которую нужно заменить, и строки, на которую она будет заменена. Эти правила записываются последовательно, каждое на новой строке.

Через -> описывается простая замена, а через => — заключительная замена.

Через // в начале строки описывается комментарий.

Поделиться ссылкой можно только на программу находящуюся в данный момент в поле ввода. Если необходимо поделиться списком программ, скачайте файл содержащий этот список.
